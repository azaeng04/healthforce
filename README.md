## Requirements
* Docker
* Node js
* Application is running locally

### Local execution
1. Navigate to the e2e folder
2. Execute `npm install`
3. Execute `npm test`

### Docker execution
1. In the root folder execute `docker compose up --build`

### Interactive mode
1. Navigate to the e2e folder
2. After an npm install execute `npm run ui`
